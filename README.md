# LocalDeviceMonitor - BLE + WIFI + LAN + MODBUS + ONVIF + BACNET

Projeto WPF em C# / .NET 8 com 3 opções de scan:
- BLE real
- WIFI real (redes visíveis via netsh)
- LAN real (ping + arp)
- MODBUS
- BACNET
- ONVIF
  
 
## Como abrir
1. Extrair o ZIP
2. Abrir `LocalDeviceMonitor.sln`
3. Abrir no Visual Studio 2022
4. Fazer Restore NuGet se for pedido
5. Carregar F5
